📱 Input dari HP via AppSheet

1. Buka appsheet.com → login dengan akun Google yang sama dengan Sheets
2. Create → App → Start with existing data → pilih spreadsheet ini
3. AppSheet otomatis baca sheet Santri & Transaksi
4. Set kolom Foto bertipe Image, kolom NIS sebagai Key
5. Tambah view Form untuk input santri + Scan (QR/Barcode) yang mencari berdasarkan NIS
